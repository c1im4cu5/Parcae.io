Stock and Crypto Predictions

Nostradamas is a machine learning algorithm designed to learn from implementing the R Squared Mean mathmatical equation against a dataset. After learning fom the dataset, the prediction bot will predict future values of a commodity based on prior performance and the R Squared Mean mathmatical equation. Users can utilize the free version (not giving any future predictions) to better understand how the machine will learn from a set of given parameters. The paid version will offer users the ability to better refine epoch iterations and choose an amount of future predictions.

Nostradamas and Gann

Gann lines have been known to assist in foretelling events. By calculating gann lines, Nostradamas can show crossover events. As you can see from the picture, crossover events will help assist in better decision making capabilities for you.

Gann lines are only available when a user supplies additional information in their dataset. In order to run a base prediction, Nostradamas only requires a dataset with registered date/time (UTC format) and closing price of the commodity. In order to interpolate Gann, a user must provide more information (Opening, Closing, High, Low, Date/Time). See guide to uploading.
